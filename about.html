About TaskFlow

What is TaskFlow?

TaskFlow is a lightweight task-management web application designed to help individuals and teams stay organized. It allows you to create, track, and manage tasks with priorities and deadlines — all from a clean, simple interface. Built as part of the DevOps CI/CD assignment, TaskFlow demonstrates automated deployment using GitHub Actions, Docker, and Render.

How to Use TaskFlow

PageWhat it Does
Dashboard See a summary of all tasks (total, pending, completed) and quick navigation links.
Add Task Fill out the form to create a new task with a title, description, due date, and priority.
Task List Browse all tasks. Use the filter buttons to show All, Pending, or Completed tasks.
Task Detail View full details of a task. Mark it as complete or delete it from here.
About This page — describes the app and explains how to navigate it.
